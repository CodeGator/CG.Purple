    Interface ISmtpClient

    This interface represents a .NET SmtpClient.

    Namespace: CG.Purple.Providers.Smtp.Internal
    Assembly: CG.Purple.Providers.Smtp.dll
    Syntax
    public interface ISmtpClient
    Remarks

    The purpose of this interface is for testability. In short, the Microsoft SMTP client is simply not mockable, at all. Or at least, it isn't mockable in any way that I understand. If you know a way to mock the SmtpClient type, please let me know.

    This type wraps a standard SmtpClient instance and defers to that object at runtime. During testing, we can then mock up the ISmtpClient type, and use that, instead.
